# Minecraft Zeus - Projeto para estudo acadêmico 
Este projeto tem como objetivo desenvolver um clone do Minecraft para fins acadêmicos/educacionais para por em prática os conhecimentos adquiridos na matéria de Programação orientada a objetos do curso de Análise e desenvolvimento de sistemas da Universidade Santa Cecília.

# Tecnologias a serem utilizadas: 
- Linguagem C#
- Unity Engine
